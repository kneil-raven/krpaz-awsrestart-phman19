1. START
2. INPUT what type of operation to be used. Either Add(+),
Subtract(-), Multiply(*), Divide(/).
3. CHECK the menu or list of operators, if the operator 
is valid.
4. IF operator is valid, go to STEP 6.
5. ELSE, PRINT operator is invalid and go back to STEP 2.
6. INPUT num1 and num2 as a number.
7. CHECK if num1 and num2 is valid data type value.
8. IF num1, num2 a valid data, go to STEP 10.
9. ELSE, PRINT num1 or num2 is invalid and go back to 
STEP 2.
10. IF valid operator is Add(+) in STEP 4, THEN 
calculate = num1 + num2.
11. ELIF valid operator is Subtract(-) in STEP 4, THEN
calculate = num1 - num2.
12. ELIF valid operator is multiply(*) in STEP 4, THEN 
calculate = num1 * num2.
13. ELIF valid operator is divide(/) in STEP 4, THEN 
calculate = num1 / num2.
14. IF num2 is 0 in STEP 13, PRINT Division by Zero Error THEN  
proceed to STEP 16.
15. PRINT calculate
16. CHECK if the user wants another calculation.
17. IF Yes, go back to STEP 2. 
18. IF No, PRINT Done Calculating.
19. END
